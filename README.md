# 🥌 Кёрлинг Онлайн

Бесплатная браузерная игра в кёрлинг с физикой льда, ИИ-соперником и режимом для двух игроков.

**[Играть прямо сейчас →](https://iskanderdzebarov-bot.github.io/curling/)**

---

## Описание

Классический кёрлинг в браузере — без установки, без регистрации. Выбирай позицию, задавай направление и контролируй силу броска. Камни скользят по льду с реалистичной физикой, а победитель определяется по итогам 4 эндов.

## Особенности

- **Два режима игры** — Игрок против Игрока (на одном устройстве) или Игрок против ИИ
- **Физика скольжения** — камни взаимодействуют друг с другом при столкновениях
- **Трёхэтапный бросок** — позиция → направление → сила
- **Система очков** — попадание в центр («пуговицу») даёт 100 очков, следующие кольца — 50, 25 и 10
- **4 энда** — полная партия, как в настоящем кёрлинге
- **Аркадный саундтрек** — синтезированная музыка прямо в браузере
- **Адаптивный интерфейс** — работает на ПК, планшетах и смартфонах (поддержка тач-экранов)

## Как играть

Каждый бросок состоит из трёх шагов:

1. **Позиция** — нажми, чтобы выбрать боковое положение для броска
2. **Направление** — нажми, чтобы зафиксировать угол
3. **Сила** — нажми в нужный момент, чтобы задать мощность броска

Камень улетит туда, куда ты прицелился. Побеждает тот, кто наберёт больше очков за 4 энда.

## Технологии

- Чистый HTML5, CSS3 и JavaScript — без фреймворков и зависимостей
- Рендеринг через Canvas API
- Работает в любом современном браузере с поддержкой JavaScript

## Запуск локально

Достаточно открыть файл `index.html` в браузере — никакой сборки не требуется.

```bash
# Или через локальный сервер, например:
npx serve .
```
